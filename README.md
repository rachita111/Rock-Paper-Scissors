# 🎮 Rock Paper Scissors

A browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, keeps track of scores, and stores game progress using the browser's localStorage.

## Features

- Play against a computer with randomly generated moves
- Displays game result after every round
- Tracks wins, losses, and ties
- Persists score using localStorage
- Reset score functionality
- Responsive and interactive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage
